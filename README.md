# RECETAS

Este proyecto fue generado usando [Angular CLI](https://github.com/angular/angular-cli) version 20.3.5.

## Servidor de desarrollo

Para iniciar un servidor de desarrollo local, ejecuta:

```bash
ng serve
```

Una vez que el servidor esté en funcionamiento, abre tu navegador y navega a http://localhost:4200/. La aplicación se recargará automáticamente cada vez que modifiques alguno de los archivos fuente.

## Scaffolding de código

Angular CLI incluye potentes herramientas de scaffolding de código. Para generar un nuevo componente, ejecuta:

```bash
ng generate component component-name
```

Para una lista completa de los esquemáticos disponibles (tales como components, directives o pipes), ejecuta:

```bash
ng generate --help
```

## Compilación (Building)

Para compilar el proyecto, ejecuta:

```bash
ng build
```

Esto compilará tu proyecto y almacenará los artefactos de compilación en el directorio dist/. Por defecto, la compilación de producción optimiza tu aplicación para rendimiento y velocidad.

## Ejecución de pruebas unitarias

Para ejecutar las pruebas unitarias con el ejecutor de pruebas Karma, usa el siguiente comando:

```bash
ng test
```
